# For Us.

A cinematic, emotionally immersive Valentine's landing page.

## View

Open `index.html` in any modern browser. No build step or server required.

## Features

- Animated gradient hero with radial glow
- Multi-layer floating particle system
- Film grain & vignette overlays
- Scroll-triggered reveal animations
- Vertical timeline with alternating memory cards
- Letter section with drop cap & parchment texture
- Ambient piano via Web Audio API
- Elegant mute toggle
- Fully responsive

## Typography

- **Cormorant Garamond** — Titles & headings
- **Lora** — Body text
- **Inter** — Subtitles & labels
